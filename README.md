# Global weather & population analysis
## Overview
This project analyzes weather conditions accross maajor cities worldwide by combining real-time weather data collected from OpenWeather API with population data from web scraping

## Project objectives
- Collect weather information from multiple global cities
- Gather city population data from public available web source
- Combine and clean datasets
- Create an interactive dashboard for reporting

## Data sources
1.Weather data: OpenWeather API (City, Country, Temperature, Humidity, Wind Speed, Weather Condition)
2.Population data: Worldometers (City Rank, City Name, Country, Population)

## Key insights
- Major Southeast Asia cities generally exhibit higher humidity levels.
- The highest temperature cities in the dataset are primarily located in tropical and desert region
- Cloudy weather conditions tend to have higher average humidity than clear conditions

## Final conclusion
This project demonstrate data analytics workflow, including data collection, data cleaning, exploratory analysis and interactive dashboard development using Power BI. The analysis showed the differences in temperature and humidity accross major cities worldwide. Additionally, population size did not show a clear relationship with temperature or humidity.
